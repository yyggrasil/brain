## Ferramentas utilizadas
- Xampp ([[PHP]]-8)
- [[Node.js]]
- [[Composer]]
- [[Postman]]
- [[React]]
- Laravel
## Instalação

criar projeto laravel
```
composer create-project --prefer-dist laravel/laravel api
```
Para rodar
```
php artisan serve
```

criar app react
```
npx create-react-app app
```
